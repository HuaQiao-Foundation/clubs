# Development Journal Template - Georgetown Rotary

## Standard Dev Journal Entry Format

When CEO requests **"Create dev journal entry"**, use this template:

```markdown
=== GEORGETOWN ROTARY DEV JOURNAL ===

**Date:** [Current date]
**Project:** Georgetown Rotary Speaker Management System
**Task:** [Brief task description]
**Status:** [Complete/Partial/Issues/Blocked]
**CTO:** Claude Code | **CEO:** [Your name]

---
```

## Usage Instructions

**CEO Command:** Simply say "Create dev journal entry" after any implementation session.

**File Storage:** Save in `/docs/dev-journals/` with format: `YYYY-MM-DD-task-description.md`

**Business Purpose:** 
- Track technical progress with business impact focus
- Maintain strategic oversight of development direction
- Document decision rationale for future reference
- Ensure all technical work serves Georgetown Rotary's mission
