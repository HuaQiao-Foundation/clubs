# Exports Directory

This directory contains generated CSV/XLSX reports and data exports from the Georgetown Rotary application.

## Naming Convention

Files should follow the pattern: `{description}-{YYYY-MM-DD}.{ext}`

Example: `georgetown-officers-and-chairs-2025-10-09.csv`

## Contents

- `georgetown-officers-and-chairs-2025-10-09.csv` - Complete list of all officer and committee chair positions with current assignments

## Note

This directory should be included in `.gitignore` if exports contain sensitive member data.
